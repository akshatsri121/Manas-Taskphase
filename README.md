# Project Manas: Custom-Power-Distribution-Board

![Status](https://img.shields.io/badge/Status-Ready_for_Manufacturing-brightgreen)
![KiCad](https://img.shields.io/badge/Designed_in-KiCad_8.0-blue)
![Architecture](https://img.shields.io/badge/Architecture-Buck_Converter_%2B_LDO-orange)

A custom, high-current Power Distribution Board (PDB) designed specifically for a differential drive robotics platform. This board takes a raw battery input and cleanly steps it down into three isolated, logic-level power rails (5V, 3.3V, 1.8V) while actively monitoring system-wide power consumption in real-time.
